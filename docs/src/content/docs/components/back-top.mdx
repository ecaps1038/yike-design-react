## API

默认样式，距离底部和右侧 `50px`，可通过 `style` 进行覆盖

| 参数            | 描述              | 类型                                | 默认值            |
|---------------|-----------------|-----------------------------------|----------------|
| type          | 默认按钮的类型         | `'primary' \| 'secondary'`        | `'primary'`    |
| style         | 样式设置            | ` React.CSSProperties`            | -              |
| visibleHeight | 滚动时触发显示回到顶部的高度	 | `number`                          | 400            |
| behavior      | 滚动时的过渡效果	       | `'auto' \| 'instant' \| 'smooth'` | `'smooth'`     |
| target        | 监听滚动的元素	        | `() => HTMLElement \| Window`     | `() => window` |
| onClick       | 点击按钮时触发         | `() => void`                      | -              |
| onChange      | 按钮显隐状态切换时触发     | `(visible?: boolean) => void`     | -              |
