Button 按钮

按钮类型 Type

按钮有三种类型：主按钮、次要按钮、线框按钮。主按钮在同一个操作区域建议最多出现一次。

import { Button, Space } from '@yike-design/react';

export default () => {
  return (
    <Space>
      <Button>主要按钮</Button>
      <Button type="secondary">次要按钮</Button>
      <Button type="outline">线框按钮</Button>
    </Space>
  );
};

按钮尺寸 Size

按钮分为：s、m、l、xl 四种尺寸。高度分别为：24px、32px、36px、48px。默认尺寸为l。

import { Button, Space } from '@yike-design/react';

export default () => {
  return (
    <Space>
      <Button size="s">按钮</Button>
      <Button size="m">按钮</Button>
      <Button size="l">按钮</Button>
      <Button size="xl">按钮</Button>
    </Space>
  );
};

按钮形状 Shape

按钮分为默认形状、circle 圆形、round 全圆角、square 正方形四种形状。

import { Button, Space } from '@yike-design/react';

export default () => {
  return (
    <Space>
      <Button>默认形状</Button>
      <Button shape="round">全圆角</Button>
      <Button shape="circle">+</Button>
      <Button shape="square">+</Button>
    </Space>
  );
};

通栏 Block

通过设置 long 属性，使按钮的宽度跟随容器的宽度。

import { Button, Space } from '@yike-design/react';

export default () => {
  return (
    <Space direction="vertical" block>
      <Button long>主要按钮</Button>
      <Button
        type="secondary"
        long
      >
        次要按钮
      </Button>
      <Button
        type="outline"
        long
      >
        线框按钮
      </Button>
    </Space>
  );
};

API

通过设置 Button 的属性来产生不同的按钮样式，推荐顺序为：type -> size -> shape -> status -> disabled。

status按钮的状态'primary'｜'success'｜'warning'｜'danger'-
