Space 间距

基础用法

使元素之间产生间隔，使其不至于离得太近，提升观感。

import { Space } from '@yike-design/react';

export default () => {
  return (
    <Space>
      <div className="demo-box-1"></div>
      <div className="demo-box-1"></div>
      <div className="demo-box-1"></div>
    </Space>
  );
};

间隔尺寸 Size

内置 4 个尺寸，分别是 s - 4px，m - 8px，l - 16px，xl -24px，默认为 l，为数字或枚举值时，是水平和垂直间距；为数组时，是 [水平间距, 垂直间距]。

对齐 Align

内置 4 种对齐方式，分别为 'start'｜'center'｜'end'｜'baseline' ，在水平模式下默认为 start。

默认上对齐

默认上对齐

默认上对齐

import { Space } from '@yike-design/react';

export default () => {
  return (
    <>
      <h4>默认上对齐</h4>
      <Space>
        <div className="demo-box-1"></div>
        <div className="demo-box-2"></div>
        <div className="demo-box-3"></div>
      </Space>
      <h4>默认上对齐</h4>
      <Space align="center">
        <div className="demo-box-1"></div>
        <div className="demo-box-2"></div>
        <div className="demo-box-3"></div>
      </Space>
      <h4>默认上对齐</h4>
      <Space align="end">
        <div className="demo-box-1"></div>
        <div className="demo-box-2"></div>
        <div className="demo-box-3"></div>
      </Space>
    </>
  );
};

方向 Direction

可以设置横排或竖排，默认横排：'vertical'｜'horizontal'。

import { Space } from '@yike-design/react';

export default () => {
  return (
    <Space direction="vertical">
      <div className="demo-box-1"></div>
      <div className="demo-box-1"></div>
      <div className="demo-box-1"></div>
    </Space>
  );
};

换行 Wrap

设置 wrap 来允许在空间不足时换行。

import { Space } from '@yike-design/react';

export default () => {
  return (
    <Space wrap>
      <div className="demo-box-4"></div>
      <div className="demo-box-4"></div>
      <div className="demo-box-4"></div>
      <div className="demo-box-4"></div>
      <div className="demo-box-4"></div>
      <div className="demo-box-4"></div>
    </Space>
  );
};

API

参数描述类型默认值
size间距尺寸's'｜'m'｜'l'｜'xl'｜number｜[number, number]'l'
align对齐'start'｜'center'｜'end'｜'baseline''start'
wrap换行booleanfalse
direction方向'vertical'｜'horizontal''horizontal'
block显示为块级元素booleanfalse
